# 🌤️ Weather Dashboard

A modern and responsive Weather Dashboard built using HTML, CSS, and JavaScript. The application fetches real-time weather data from a public REST API using Fetch API and Async/Await.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Display temperature
- 💧 Show humidity
- 💨 Display wind speed
- 🌍 Real-time weather data
- ⚡ Built with Fetch API & Async/Await
- ❌ Error handling for invalid city names and network issues
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Async/Await
- REST API
- JSON

## 📚 Learning Outcomes
